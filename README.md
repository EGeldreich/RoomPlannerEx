## Getting Started

### Requirements
-  Node.js version 14 or higher
-  NPM (comes with Node.js)

### Setup
1. Clone this project:
```bash
git clone https://github.com/EGeldreich/RoomPlannerEx
cd RoomPlannerEx
```

2. Start the server:
```bash
node server.js
```

3. Visit http://localhost:3000 in your browser
